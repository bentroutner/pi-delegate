# Simple Test Task for Pi

## Objective
Create a simple "Hello World" Node.js program to test the pi-delegate workflow.

## Requirements

1. Create a file at `/home/ben/pi-test/hello.js`
2. The program should:
   - Print "Hello from Pi!" to console
   - Print the current date and time
   - Exit with code 0

3. Use ESM format (no CommonJS)
4. Follow the node.js-style conventions:
   - 2 spaces indentation
   - No semicolons
   - Single quotes
   - const/let only

## Testing

After creating the file, run it with:
```bash
node /home/ben/pi-test/hello.js
```

Report back:
1. Did the program run successfully?
2. What was the output?
3. Any issues?
